# MapSaver 🌍 (中文介绍)

[English](./README.md) | **简体中文**

MapSaver是一款轻量级的地图可视化与高清导出工具。内置中国行政区划数据，支持从全国到市级的交互式下钻。并可通过上传 GeoJSON 文件或粘贴文本直接渲染地图，实时调整颜色、标签及边框样式，一键生成高清图片（PNG/SVG）。

## 功能特性 ✨

- **内置快捷地图** - 内置全国省市地图，支持点击下钻及平滑的面包屑导航，点击地图空白处可快速返回上一级，操作高效便捷。
- **自定义 GeoJSON 源** - 一键上传本地 GeoJSON 文件或粘贴数据代码，让地图渲染摆脱限制。
- **外观高度定制** - 自由配置地图背景、区块颜色、边框颜色及边框宽度。支持显示/隐藏地图地名标签，调整标签配色与大小。
- **高阶导出能力** - 支持导出高达 4x 的超清 PNG 高清长图，或者无限放大不失真的纯 SVG 矢量图。
- **离线支持** - 即使断开网络，已访问过的应用依然可离线使用或作为独立应用安装。

## 技术栈 🛠️

- **Vite** - 极速的下一代前端构建工具
- **Echarts 5** - 强大的开源可视化图表库
- **Coloris** - 优雅轻量化的取色器组件
- **原生 JS/CSS** - 原生 JS/CSS，无沉重框架负担，极速轻简。

## 快速运行 🚀

### 环境要求

请确保你的电脑上已经安装了 [Node.js](https://nodejs.org/) 环境。

### 初始化安装

1. 克隆代码库
   ```bash
   git clone https://github.com/yourusername/MapSaver.git
   ```
2. 进入工程目录
   ```bash
   cd MapSaver
   ```
3. 安装关联依赖
   ```bash
   npm install
   ```

### 本地开发

运行以下命令唤起本地调试服务器：

```bash
npm run dev
```

并在浏览器中打开 [http://localhost:5173](http://localhost:5173)。

### 生产环境构建

执行编译打包命令：

```bash
npm run build
```

最后的成品将被输出至 `dist` 目录下。

### 如何使用 💡

1. **选择地图源**：在左侧“数据源”面板中，从“内置地图”选择省市，或切换到“上传/粘贴 JSON”导入你的本地地图数据。
2. **深度交互**：在地图区域点击省份或城市，即可直接下钻查看下级行政区，通过上方“面包屑”导航随时回退。
3. **个性化配置**：通过左侧“样式配置”面板，实时调整地图背景、区域填充色、边框粗细及文本标签样式。
4. **一键导出**：在“导出配置”面板选择清晰度（1x - 4x）与格式（PNG/SVG），点击“导出地图”按钮即可保存你的地图。
